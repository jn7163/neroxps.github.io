遇到的坑

数据库编码格式导致中文输入字符无法保存

由于之前使用的 docker 创建的 zabbix 数据库默认是以latin1字符编码创建的，故此在问题输入『消息』的时候会报字符编码错误。

那么数据库里面已经有原始数据存在，所以我需要将数据导出，再将数据转换成utf8编码。

参考：# MySQL数据库从GBK转换到UTF-8最简单解决方案（也适用于其它编码转换）
